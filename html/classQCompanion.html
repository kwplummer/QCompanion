Detailed Description

The main class of the program.

This class organizes all the components, schedules when speaking, notifying, and screenshot-logging will occur.

void QCompanion::speak ( )

Reads components.

Looks at each component, and if it wants to be read, it sends it to the Speaker. It also looks for the component that wants to be read earliest, and schedules the next call to speak().
