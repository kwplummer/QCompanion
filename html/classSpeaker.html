void Speaker::readLoop ( )
private

The main loop used for speaking and sending notifications.

The main loop that the speaker runs. While stopReading is false the loop waits for a string to be added to the queue, it pops it out, and then sends it to flite (if canSpeak is enabled) as well as to libnotify (if canSendNotifications are enabled).
