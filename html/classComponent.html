Detailed Description

Everything that has something to say derives from this.

This abstract class provides an interface for all parts of the program that speak or notify. Also it allows for it to be muted so that they will be excluded from speaking. It provides a list of menu options, that can be overridden as well as nextCheckTime() which indicates when the component wants to speak next, as well as getText(), which returns the message to speak/show.

virtual QDateTime Component::nextCheckTime ( )
pure virtual
Note
While nextCheckTime() is not const (you can change state), it may be called many times before Speaker::speak is called. You should not do something like increment and return a time by 60 seconds, as you'll never get called. However, the last value returned is always checked before each call of nextCheckTime().

Implemented in QSnapper, WaiterComponent, and HourReader.
